# NeuralPlayground
NeuralPlayground is an unity evolution simulation based on something like NEAT.

## Basic idea
- You can create your own enviomnt and then spawn population and let it learn to survive.

## What is already done
- You can spawn population with randomly inicialized neural network
- They learn to eat food so they don't die
- They can crossbreed
- There are implemented basic mutations to weights and biases
